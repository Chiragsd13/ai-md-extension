How It Works

AI.md saves your project context (git state, open files, tech stack, task notes) as a private Gist on GitHub. On any device or AI platform, click @ AI.md in the chat to resume right where you left off.


Workflow:

  1. Install the VS Code extension or CLI (npm i -g aimd)
  2. Save context: aimd save or Ctrl+S in VS Code
  3. Open Claude / ChatGPT / Gemini in any browser
  4. Click the @ AI.md button or use the toolbar popup
  5. Your full project context is injected — continue seamlessly
