How It Works

AI.md saves your project context (git state, open files, tech stack, task notes) to your preferred cloud storage. On any device or AI platform, click @ AI.md in the chat to resume where you left off.


Smart features:

  • Auto-tracking — watches your conversation, scores each exchange for importance, batches saves
  • AI-powered update — asks the AI you're chatting with to write the context doc (no extra API cost)
  • Auto-save triggers — tab close, focus lost, battery low, token limit warning
  • Multi-cloud — GitHub Gist, Google Drive, or local download

Works with: VS Code, Cursor, JetBrains IDEs, CLI (npm i -g aimd), Chrome extension
